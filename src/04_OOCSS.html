OOCSS

Object Oriented CSS - Patrón que se repite visualmente, puede ser abstraido a un componente y ser reutilizado

  • Separación entre estructura y presentación
  • Separación entre contenido y contenedor

SMACSS

Scalable and Modular Architecture for CSS - Más una guía de estilo que un framework, define una serie de convenciones para el nombrado de clases y la organización de los archivos de CSS

Base:
Estilos por defecto y transversales
Layout:
Secciones de un site como por ejemple el Header, las clases llevan el prefijo l- para identificarlos
Modules
Componentes reutilizables incluidos dentro de elementos de layout u otros modulos
State
Estilos para cambios de estado causados principalmente por interacciones con el usuario, las clases se nombran con el prefijo is- seguido del estado
Theme
Estilos que forman el aspecto visual
https://smacss.com

BEM

Block Element Modifier es un enfoque basado en componentes. La idea es dividir la interfaz en bloques independientes que se puedan combinar y reutilizar

        
            <ul class="nav-menu">
                <li class="nav-menu__item" >
                    <a></a>
                </li>
                <li class="nav-menu__item">
                    <a></a>
                </li>
                <li class="nav-menu__item nav-menu__item--active">
                    <a></a>
                </li>
            </ul>
Block:

Componente individual que puede ser reutilizado, tiene sentido por sí mismo

                
                    .nav-menu {
                        /* Main nav styles */
                    }
Element

Parte de un bloque que no tiene sentido por sí mismo y está semánticamente atado al bloque

                
                    .nav-menu {
                        /* Main nav styles */
                    }

                    .nav-menu__item {
                        /* Item inside main nav styles */
                    }
Modifier

Representa un estado de un bloque o un elemento, se usa para representar apariencia o comportamiento

                
                    /* Hyphenated BEM */

                    .nav-menu__item--active {
                        /* Active item styles */
                    }

                    /* Strict BEM (Bastante menos habitual) */

                    .nav-menu__item_state_active {
                        /* Active item styles */
                    }
Mix

Permite combinar Bloques en un nodo para reutilizar estilos y crear nuevos componentes

                

                    .nav-menu .header__nav-menu {
                        /* Active item styles */
                    }

ITCSS

Inverted Triangle CSS es una forma de organizar los archivos CSS para tratar con aspectos específicos de CSS como espacio de nombres global, la cascada y la especifidad

Settings:
Definiciones de fuentes, colores, tamaños, etc...
Tools
Mixins y funciones utilizados globalmente
Generic
Resets, normalize, definiciones de box-sizing... Es la primera capa que genera CSS
Elements
Estilos aplicados a los elemntos HTML
Objects
Selectores basados en clases que definen patrones, por ejemplo el Media Object
Components
Elementos específicos de la interfaz de usuario que habitualmente extienden objetos
Utilities
Clases de ayuda con la habilidad de sobreescribir cualquier cosa anterior en la piramide
