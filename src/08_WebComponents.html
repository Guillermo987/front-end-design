Web Components

Host

        const menu = document.createElement('nav');

Elemento al que se le incorpora un fragmento de documento.


        element.attachShadow({mode: 'open'});
Shadow root:

Fragmento de documento que se incorpora al host.

Light DOM:

El marcado que el usuario del componente crea, existe fuera del Shadow DOM. Son los hijos del elemento.

Shadow DOM:
El DOM local del componente, define su estructura, ámbito de estilos (Scope) y encapsula los detalles de implemnetación. Puede definir como renderizar el marcado que el consumidor del componente incorpora.
Slots:
Son la forma de incorporar el marcado del usuario del componente con el marcado del componente(componer Light DOM con Shadow DOM)
CSS Custom Properties

Variables nativas de CSS con las que se puede acceder al Shadow DOM 

        
        html {
            --primary-color: #ff9500;
        }

        .element {
            color: var(--primary-color);
        }
:host

Selector para el elemento en el que se incorpora el Shadow Dom, los estilos definidos, son accesibles desde fuera 

        
        :host {
            display: block;
        }
:host()

Permite especificar id, clase o atributo del propio elemento

        
        :host([hidden]) { 
            display: none 
        }
:host-context()

Permite especificar id, clase o atributo de cualquier elemento padre 

        
        :host-context(.green-theme) { 
            background-color: green; 
        }
::sloted()

selector que permite acceder a los elementos de un slot, sólo para elementos de primer nivel 

        
        ::sloted(h1) { 
            font-size: 3rem;
        }
