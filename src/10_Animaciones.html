Animaciones

Transiciones

Animaciones por transición del valor del estado de una propiedad CSS

transition-property
Propiedad css que va a realizar la transición
transition-duration
Duración de la animación
transition-timing-function
Función que va a definir como la animación va a distribuirse durante la duración de la transición se sus estados
transition-delay
Tiempo que va a transcurrir hasta que la animación inicie
.animate { transform: translateX(-100%); transition: transform 0.5s ease-in-out 0.5s; } .animate--end { transform: translateX(0); }

Keyframes

Controla los estados intermedios de la secuencia de animación.

animation-name
Nombre que se le va a asignar a a la animación
animation-duration
Duración de la animación
animation-timing-function
Función que va a definir como la animación va a distribuirse durante la duración de la transición se sus estados
animation-delay
Tiempo que va a transcurrir hasta que la animación inicie
animation-iteration-count
Número de veces que se va repetir la animación
animation-direction
Dirección en la que va a transcurrir la animación (normal, reverse, alternate, altrnate-reverse)
animation-fill-mode
Como la animación va a aplicar estilos a su target antes y después de la transición
animation-play-state
Detiene o reanuda una animación, cuando la reanuda, la animación prosigue desde el estado que se detuvo.
    
    @keyframes spin {
        0% {
            transform: rotate(0deg);
        }
    
        50% {
            transform: rotate(180deg);
        }
    
        100% {
            transform: rotate(359deg);
        }
    }
    
    .spinner {
        animation-name: spin;
        animation-duration: 3s;
        animation-timing-function: linear;
        animation-iteration-count: infinite;
    }

Rendering path

Layout:
El navegador determina cuanto espacio necesita cada elemento y donde lo coloca
Painting:
En este proceso se dibujan los elementos y se rellenan los píxeles
Compositing:
El navegador dibuja los elementos en el orden correcto y se renderiza la página
will-change
Propiedad CSS que indica al navegador que una propiedad va a cambiar y pueda realizar optimizaciones anticipando recursos, usar con moderación como último recurso para resolver problemas de rendimiento, no anticiparlos.
