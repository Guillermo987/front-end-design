Responsive Web Design

Responsive Web Design es un término creado por el diseñador Ethan Marcotte para referirse al conjunto de tecnicas que utilizaba para hacer que una misma página pudiera visualizarse en dispositivos con distintas dimensiones de pantalla.

Retícula flexible:
Tradicionalmente se creaban con la propiedad float y medidas en porcentaje, ahora lo habitual es utilizar Flexbox y ya se empiezan a ver implementaciones con Grid.
Imágenes flexibles:
                
                    .responsive-img {
                        display: block;
                        max-width: 100%;
                    }
Media queries:
                
                    @media (min-width: 20em) and (max-width: 46.24em) {
                        .container {
                            /* Tablet layout */
                        } 
                    }
                        
                    @media (min-width: 46.25em) {
                        .container {
                            /* Desktop layout */ 
                        } 
                    }

Otros elementos con naturaleza flexible

Unidades de medida relativas

El navegador permite al usuario adaptar el tamaño de la fuente, si utilizamos medidas relativas el diseño se sostendrá aun cuando el tamaño de fuente se modifique.

  • Para cumplir el requerimiento de accessibilad de poder ampliar el tamaño de la fuente hasta un 200%
  • Para cumplir el requerimiento de poder ampliar el diseño en un 100%
em:
Unidad relativa al font-size del padre más cercano al elemento, el cual que el contexto. 
            
              /* Mala práctica muy extendida para facilitar el cálculo de los em */

              html {
                font-size: 62.5%
              }

              /* Práctica recomendable */

              html {
                font-size: 100%; // Suelen ser 16px por defecto
              }

              body {
                font-size: 1em; // 16px
              }

              ul {
                font-size: 0.75em; // 12px
              }

              li {
                font-size: 1em; // Relativo al elemento ul 12px
              }
rem (root em):
Unidad relativa al elemento raiz del documento, <html> que define el contexto. 
            
                html {
                  font-size: 100%; // Suelen ser 16px por defecto
                }
                body {
                  font-size: 1rem; // 16px
                }
  
                ul {
                  font-size: 0.75rem; // 12px
                }
  
                li {
                  font-size: 1em; // Relativo al elemento root 16px
                }

Para calcular el tamaño de la fuente relativa, dividir el tamaño de la fuente que se quiere obtener en pixels entre el contexto.

Es de gran ayuda el inspector de propiedades computadas del navegador, ya que muestra las unidades en pixels

SVG

Scalable Vector Graphic, formato vectorial de imagen basado en XML para gráficos bidimesionales con soporte para interacción y animación. Es un estándar abierto desarrollado por la W3C desde 1999. Tiene amplio soporte en los principales navegadores.

Formas de incorporarlo en el documento:

  • Imagen: 
                    
                    <img class="svg-icon" src="img/icon.svg" alt="icon">
  • Imagen de fondo: 
                    
                    .svg-img {
                        background-image: url(images/icon.svg);
                    }
  • En línea: 
                
              <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24">
                  <path clip-rule="evenodd" fill="none" d="M0 0h24v24H0z"/>
                  <path d="M22.7 19l-9.1-9.1c.9-2.3.4-5-1.5-6.9-2-2-5-2.4-7.4-1.3L9 6 6 9 1.6 4.7C.4 7.1.9 10.1 2.9 12.1c1.9 1.9 4.6 2.4 6.9 1.5l9.1 9.1c.4.4 1 .4 1.4 0l2.3-2.3c.5-.4.5-1.1.1-1.4z"/>
              </svg>

ViewBox

  • Es el lienzo que define que parte del SVG es visible, define los valores, coordenada x e y del gráfico respecto al viewBox seguidos del ancho y alto de este lienzo.
  • El viewBox define las proporciones del gráfico, que pueden ser manipuladas usando el atributo preserveAspectRatio.
