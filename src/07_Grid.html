Grid Layout Module

Sistema de retícula para distribuir el espacio y alinear el contenido.

  • Relación directa entre el contenedor padre y sus descendientes directos.
  • Dos dimensiones, fila y columna
  • Permite adaptar el tamaño del contenido a las dimensiones del contenedor.
  • Permite alterar el orden visual de los elementos.

Terminología

Grid Container:
El elemento que define el grid.
Grid Item:
Descendiente de primer nivel, elemento HTML que sea un descendiente directo del Grid Container.
Grid Line:
Línea horizontales (fila o row) o vertical (columna o column) que separa el grid en secciones. Se referencian por número y pueden ser nombradas.
Grid Cell:
La intersección entre una fila y una columna.
Grid Track:
El espacio que forma dos o más Grid Lines adjacentes. Row Tracks cuando las líneas son horizontales o Row Columns cuando son verticales.
Grid Area:
El area rectangular formado por cuatro Grid Lines específicas que pueden abarcar una o más Grid Cells, definen un área dentro del Grid.
Grid Gap:
Espacio vacío entre Grid Tracks.
  • Cualquier Grid Item puede ser colocado en cualquier Grid Cell o Grid Area.
  • Se pueden colocar unos elementos encima de otros, por defecto se respeta el orden del código, se pueden reorganizar utilizando la propiedad z-index.
  • La colocación manual no es afectada por el orden del código.
  • Se pueden anidar Grids.
  • Se pueden aplicar condicionalmente.
  • Se pueden usar en media queries.
  • Se pueden añadir, modificar o eliminar sin afectar el HTML.
  • Los descendientes de segundo nivel necesitan declarar su propio Grid Container.

Comprueba el soporte del navegador

Feature query:

    
    @supports (grid-area: auto) {

    }

Define un Grid


    .main-grid {
        display: grid;
        grid-template-columns: 1fr 1fr 1fr;
        grid-template-rows: 1fr 1fr 1fr;
    }

Posiciona los elementos sobre las columnas del Grid

    
    .column-1 {
        grid-column-start: 1;
        grid-column-end: 2;
    }

    .column-2 {
        grid-column-start: 2;
        grid-column-end: 3;
    }

    .column-3 {
        grid-column-start: 3;
        grid-column-end: 4;
    }

Posiciona los elementos sobre las filas del Grid

    
    .row-1 {
        grid-row-start: 1;
        grid-row-end: 2;
    }

    .row-2 {
        grid-row-start: 2;
        grid-row-end: 3;
    }

    .row-3 {
        grid-row-start: 3;
        grid-row-end: 4;
    }

Shorthand

    
    grid-row: grid-column-start / grid-column-end;
    
    grid-column: grid-column-start / grid-column-end;
    
    grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end;

Nombrar grid-lines

    
    grid-template-columns: [column-1] [column-2] [column-3];
    grid-tempalte-rows: [row-1] [row-2] [row-3];

Nombrar grid-areas

    
    grid-template-area: "header header header"
                        "nav main main"
                        "footer footer foter":

    .grid-header {
        grid-area: header;
    }

    .grid-nav {
        grid-area: nav;
    }

    .grid-footer {
        gird-area: footer;
    }

Alineamiento Box Alignment Module

Propiedades aplicables al Grid Container

La propiedad justify-items alinea el contenido cada celda en el eje de línea row 


    .grid-container {
        display: grid;
        justify-items: start | end | center | stretch;
    }

La propiedad align-items alinea el contenido cada celda en el eje de bloque column 


    .grid-container {
        display: grid;
        align-items: start | end | center | stretch;
    }

La propiedad justify-content alinea el contenido del grid, si hay espacio disponible, en el eje de línea row 


    .grid-container {
        display: grid;
        justify-content: start | end | center | stretch | space-around | space-between | space-evenly;
    }

La propiedad align-content alinea el contenido del grid, si hay espacio disponible, en el eje de bloque column 


    .grid-container {
        display: grid;
        align-content: start | end | center | stretch | space-around | space-between | space-evenly;
    }

Alinear grid-items individuales:

La propiedad justify-self alinea el elemento en su respectiva celda en el eje de línea row 


    .grid-container {
        display: grid;
        justify-self: start | end | center | stretch;
    }

La propiedad align-self alinea el elemento en su respectiva celda en el eje de línea column 


    .grid-container {
        display: grid;
        align-self: start | end | center | stretch;
    }
