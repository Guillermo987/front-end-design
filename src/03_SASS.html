SASS
Sintactically Awesome Style Sheets

  • Super set de CSS compatible con el lenguaje.
  • Dos sintaxis .sass y .scss (Poco habitual ver .sass).
  • Tres implementaciones, Ruby, LibSass (Node) y Dart.
  • Es necesario una tarea de compilación a CSS con Ruby, Grunt, Gulp, Webpack…

Variables

  • Sass permite asignar valores de propiedades CSS a variables
  • También se pueden asignar otras variables y operaciones

    $black: #000000;
    $awesome-color: $black;
    $grey: lighten( $black, 10% );

    $brand-color: blue !default;

Anidación

  • Los selectores se pueden anidar respetando su correspondiente jerarquía en el maracado
  • Pseudo-clases y pseudo-elementos precedidos del símbolo &

    nav {
        ul {
            margin: 0;
            padding: 0;
            list-style: none;
        }
        
        li { display: inline-block; }
        
        a {
            display: block;
            padding: 1rem 1rem;
            text-decoration: none;

            &:hover {
                text-decoration: underline;
            }
        }
    }

Parciales e imports

  • Se puede distribuir las hojas de estilos en distintos archivos
  • El guión bajo que precede al nombre indica que el archivo es un parcial y por tanto no se compila
  • La instrucción @import se declara en el archivo en el que se quiere compilar, no es necesaria la extensión 
    
        // Estructura de archivos

        _settings.scss
        _tools.scss
        _generic.scss
        _elements.scss
        _objects.scss
        _components.scss
        _utilities.scss
        styles.scss

        // Código en styles.scss

        @import "settings";
        @import "tools";
        @import "generic";
        @import "elements";
        @import "objects";
        @import "components";
        @import "utilities";

Mixins

  • Los mixins son declaraciones de conjuntos de estilos reutilizables
  • Admiten parámetros
  • Sólo se compilan allí donde se usan, la declaración original no genera código 
      
          @mixin clearfix() {
              &::after {
                  display: block;
                  clear: both;
                  content: "";
              }
          }

          .floating-children {
              @include clearfix;
          }

Herencia

  • Existe la posibilidad de extender de una clase
        
            /* This CSS will print because %message-shared is extended. */
            %message-shared {
                border: 1px solid #ccc;
                padding: 10px;
                color: #333;
            }
            
            // This CSS won't print because %equal-heights is never extended.
            %equal-heights {
                display: flex;
                flex-wrap: wrap;
            }
            
            .message {
                @extend %message-shared;
            }
            
            .success {
                @extend %message-shared;
                border-color: green;
            }
            
            .error {
                @extend %message-shared;
                border-color: red;
            }
            
            .warning {
                @extend %message-shared;
                border-color: yellow;
            }

Operadores y funciones

  • Sass permite crear funciones con SassSript
        
            $browser-context: 16;

            @function em($pixels, $context: $browser-context) {
                @if (unitless($pixels)) {
                    $pixels: $pixels * 1px;
                }
                
                @if (unitless($context)) {
                    $context: $context * 1px;
                }
                
                @return $pixels / $context * 1em;
                }
            }
