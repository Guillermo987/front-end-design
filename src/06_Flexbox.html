Flexible Box Module

Contenedores flexibles para distribuir el espacio y alinear el contenido.

  • Relación directa entre el contenedor padre y sus descendientes directos.
  • Una dimensión, fila o columna.
  • Permite adaptar el tamaño del contenido a las dimensiones del contenedor.
  • Permite alterar el orden visual de los elementos.

Terminología

Flex container
Contenedor padre
Flex Items
Descendientes directos o de primer nivel
Main Axis
Eje principal sobre el que los elementos fluyen, por defecto es de derecha a izquierda
Main Start
Principio del eje principal desde el que los elementos empiezan a fluir
Main End
Final del eje principal hasta el que los elementos fluyen
Main Size
Longitud desde el Main Start hasta el Main End
Cross Axis
Eje perpendicualr al Main Axis
Cross Start
Principio del Cross Axis
Cross End
Final del Cross Axis
Cross Size
Longitud del Cross Axis

Propiedades del contenedor padre

Como elemento de bloque:

    
    .flex-container {
        display: flex;
    }

Como elemento de línea:


    .flex-container {
        display: inline-flex;
    }

Flujo del eje principal Main Axis

    
    .flex-container {
        display: flex;
        flex-direction: row; /* row-reverse; column; column-reverse; */
    }

Salto de línea en el eje principal

    
    .flex-container {
        display: flex;
        flex-direction: row;
        flex-wrap: nowrap; /* wrap; wrap-reverse; */
    }

Abreviatura Shorthand

    
    .flex-container {
        display: flex;
        /* flex-flow: flex-direction flex-wrap; */
        flex-flow: row nowrap;
    }

Propiedades de los descendientes directos

Orden de visualización Display Order

Orden que el elemento va a ocupar respecto a los demás flex-items 

    
    .flex-item {
        order: 0; /* -1; 1; 2; */
    }

Hay que tener consideración por la accesibilidad, las tecnologías asistenciales van a tener en cuenta el orden del código, por lo que el orden semántico no debe depender del orden visual

Flexibilidad

Factor de crecimiento Flex Grow Factor

Espacio disponible que el elemento va a ocupar en relación a los demás flex-items 

    
    .flex-item {
        flex-grow: 0; /* 1; 2; 3; ... */
    }

Factor de encojimiento Flex Shrink Factor

Factor sobre el que elemento va a encoger en relación a los demás flex-items cuando no hay espacio suficiente 

    
    .flex-item {
        flex-shrink: 0; /* 1; 2; 3; ... */
    }

Tamaño inicial Flex Basis

Dimensión inicial del elemento flex-item antes de que el espacio disponible se distribuya 

    
    .flex-item {
        flex-basis: auto /* 10px; 10%; 10em; 10rem; */
    }

Abreviatura Shorthand


    .flex-item {
        /* flex: flex-grow flex-shrink flex-basis; */
        flex: 0 0 auto;
    }

Alineamiento Box Alignment Module

Flex Container & Main Axis

Propiedad justify-content distribuye el espacio entre los elementos a lo largo del eje principal 


    .flex-container {
        display: flex;
        justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
    }

Flex Container & Cross Axis

Propiedad align-content distribuye el espacio entre los elementos a lo largo del eje perpendicular 


    .flex-container {
        display: flex;
        align-content: flex-start | flex-end | center | space-between | space-around | stretch;
    }

Propiedad align-items alinea los elementos a lo largo del eje perpendicular 


    .flex-container {
        display: flex;
        align-items: stretch | flex-start | flex-end | center | baseline;
    }

Flex Item:

Propiedad align-self permite sobreescribir el comportamiento de align-items en el elemento 


    .flex-item {
        display: flex;
        align-self: auto | flex-start | flex-end | center | baseline | stretch;
    }
