Conceptos fundamentales

Flujo normal del documento

La forma que los elementos se colocan por defecto

Elementos de bloque
Ocupan el 100% del ancho del elemento que lo contiene y el 100% del alto de su contenido
Si va seguido de otro elemento de bloque se coloca debajo del que le precede
Elementos de línea
Ocupan el 100% del ancho y alto de su contenido
Si va seguido de otro elemento de línea o texto, se coloca segido hasta llegar al ancho del contenedor, que pasaía a estar debajo
No se les puede dar dimension mediante width o height

Modelo de caja

Todo elemento del documeto tiene la estructura de una caja rectangular

Si su border-box es content-box (el valor por defecto) la propiedad width define solo el ancho del contenido

Si su border-box es border-box la propiedad width define el ancho total incluyendo el contenido, padding y bordes. Lo mismo aplica para el alto

Posicionamiento

Cuando un elemento tiene posición relativa, el espacio que ocuparía en el flujo normal del documento queda reservado, todos los elementos que hay a su alrededor se renderizan como si el elemento posicionado estuviese ocupando su flujo normal, pero si se alteran las propiedades top, right, bottom o left el elemento se desplazará acorde alos valores asignados

Cuando se posiciona un elmento absolutamente, el espacio que ocuparía en el flujo normal del documento se colapsa, el elemento se posiciona respecto a su ancestro relativo más cercano y los valores top, right, bottom o left asignados

Cuando un elemento tiene posición fija, también sale del flujo normal del documento y su espacio se colapsa, pero se posiciona respecto al viewport y los valores top, right, bottom o left asignados, si se hace scroll sobre el documento, el elemento posicionado se mantiene fijo es su posición

Cuando el elemento tien posicionamiento sticky éste se posiciona de acuerdo al flujo normal de la página con un offset relativo a su ancestro más cercano con un mecanismo de scroll
